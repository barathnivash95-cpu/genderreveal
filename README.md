## GitHub Pages Setup

1. Create a new GitHub repository, for example:
   `gender-reveal-invitation`
2. Upload all files from this folder into the repository.
3. Go to repository **Settings**.
4. Open **Pages** from the left side menu.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
6. Click **Save**.
7. GitHub will publish the site. It can take a few minutes.

Your link will usually look like:

`https://YOUR-GITHUB-USERNAME.github.io/gender-reveal-invitation/`
